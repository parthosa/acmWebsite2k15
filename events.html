ACM International Coding League
|date|

Based on the pattern of ACM-ICPC, ACM International Coding League, (abbreviated as BITSACM-ICL or just ICL ) is a computer programming competition among the universities of the world. It is organized during APOGEE, the International Annual Technical Festival of BITS, Pilani. The contest is divided into two phases. The first phase is held online. Qualifying participants are then called on campus for the final phase. The participants are expected to solve certain algorithmic problems using either of C, C++ or Java. Judging is done using an automated judge. Sample contest problems can always be viewed at http://www.codechef.com/ It was conducted in APOGEE 2011, 2012 and 2013.

Checkmate

A contest designed to test the mental aptitude of the contestants. What matters is your perseverance to get out of unforeseen situations. It consists of brain teasers, sudoku and few other mathematical questions. This is a two tier event consisting of written elims and online finals. It was conducted in APOGEE 2011, 2012 and 2013.

Its Elementary
|date|

This is a programming contest but instead of using the common programming languages like C, C++, Java we give an esoteric programming language to the participants on the spot. The participants are expected to go through the documentation of the language in one hour and then solve a number of relatively simple questions using this language. The judging, here too, is done through an automated judge (http://mooshak.dcc.fc.up.pt/). It was conducted in September 2013 and will be held this year too.
