Reduction of Bandwidth Usage using Server Client Sharing Model

By Shrey Shah Communication and Network Systems

The user sends a request to download a file from the internet. A hash code is generated by the software. It checks if the file is on the server by matching the file's hash code. If the code matches then it downloads the file. If the hash code does not match then the file is stored on the server.
